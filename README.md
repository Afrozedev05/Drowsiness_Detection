# AI Driver Drowsiness Detection System

## Overview
This project is a real-time driver drowsiness detection system using Python and OpenCV.

The system monitors the driver's eyes through a camera and alerts the driver when drowsiness is detected.

---

## Features
- Real-time face detection
- Eye monitoring
- Drowsiness alert system
- Driver presence validation
- Low-cost prototype design

---

## Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

---
